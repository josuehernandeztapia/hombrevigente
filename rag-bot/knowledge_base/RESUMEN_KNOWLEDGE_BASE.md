## 🔧 Uso en RAG ChatVigente

### Formato optimizado para:

**1. Embeddings generation**:
- Keywords específicos al final de cada archivo
- Hashtags categorizados
- Lenguaje natural para búsqueda semántica

**2. Chunking strategy**:
- Secciones bien delimitadas con headers markdown
- Tamaño secciones: 200-800 tokens (ideal para chunking)
- Metadata en encabezado (precio, categoría, fase)

**3. Context enrichment**:
- Comparaciones con alternativas (ayuda a recomendar)
- Arquetipos objetivo (personalización según cliente)
- Pricing con membresías (cálculos automáticos)

**4. Response generation**:
- Placeholders claros indican dónde NO especular
- Fuentes pendientes evitan alucinar referencias
- Tono profesional consistente
